## The Agent Lifecycle

Each round, every agent runs the **Perceive â†’ Reason â†’ Act â†’ Reflect** loop:

1. **Perceive** â€” read messages from the bus (votes, proposals, token gifts, dopamine rewards)
2. **Reason** â€” evaluate pending proposals through their strategy (rule-based for Residents, LLM-driven for Triad)
3. **Act** â€” cast votes; Triad agents may spend a token to generate a proposal or research insight
4. **Reflect** â€” update personal memory, record outcomes, update fitness score

Agents accumulate memories over sessions. Their histories inform future votes. Triad agents who run out of tokens must wait for squad placement rewards before making LLM calls again.
